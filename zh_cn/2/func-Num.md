# 内置函数 - 运算函数

## 四则运算函数

| 函数名称       | 范式                         | 作用描述                    |
| -------------- | ---------------------------- | --------------------------- |
| 加法           | `plus(x: Num, y: Num): Num`  | ···                         |
| 减法           | `minus(x: Num, y: Num): Num` | ···                         |
| 乘法           | `times(x: Num, y: Num): Num` | ···                         |
| 除法           | `div(x: Num, y: Num): Num`   | ···                         |

## 扩展运算函数

| 函数名称       | 范式                         | 作用描述                    |
| -------------- | ---------------------------- | --------------------------- |
| 取整           | `floor(x: Num): Num`         | 向下取整                    |
| 取余           | `mod(x: Num, y: Num): Num`   | 将 x, y 取整数后 计算 x % y |
