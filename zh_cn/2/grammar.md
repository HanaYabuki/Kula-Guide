# 语法

## 值

| 名称     | 值            | 应用场景                 |
| -------- | ------------- | ------------------------ |
| 无类型   | `Null`        | 无返回值，数组空值       |
| 任意类型 | `Any`         | 不确定返回值类型         |
| 数字     | `Num`         | 整数，单精度浮点数，布尔 |
| 字符串   | `Str`         | 字符，字符串             |
| 内置函数 | `BuiltinFunc` | 内置函数，扩展函数       |
| 匿名函数 | `Func`        | 函数，函数式编程         |
| 数组     | `Array`       | 定长数组                 |
| 字典     | `Map`         | K-V结构，对象，JSON      |
