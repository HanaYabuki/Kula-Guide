# 输入/输出 函数

## 常规IO （命名空间`Shell`）

| 函数名称 | 范式                         | 作用描述                      |
| -------- | ---------------------------- | ----------------------------- |
| 打印     | `print([x: Any][,]): None`   | 在控制台输出多个`x`           |
| 打印一行 | `println([x: Any][,]): None` | 在控制台输出多个`x`，最终换行 |
| 输入     | `input(): Str`               | 在控制台等待一行输入          |

## 异常 
| 函数名称 | 范式                    | 作用描述                |
| -------- | ----------------------- | ----------------------- |
| 抛出异常 | `throw(msg: Str): None` | 抛出信息为 `msg` 的异常 |
